# Analisis de condicion y precio de los vehiculso en venta 

Aplicación web desarrollada para visualizar el estado de los vehículos en venta mediante análisis exploratorio de datos.  
Incluye un histograma que muestra la distribución de las condiciones de los vehículos y un diagrama de dispersión que representa la relación entre el precio y la condición, permitiendo identificar variaciones, tendencias y posibles patrones en el mercado.
